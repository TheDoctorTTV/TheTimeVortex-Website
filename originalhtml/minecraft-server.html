Join The Time Vortex Minecraft server for a unique survival and creative experience with a friendly community from around the world. Use Bedrock or Java and enter the server IP - mc.thetimevortex.xyz - to start playing and unleashing your creativity and survival skills.

Follow this step-by-step guide to join The Time Vortex Minecraft server, regardless of whether you're using the Java or Bedrock version:

Java Version:

1. Launch Minecraft Java Edition.

2. Click on the "Multiplayer" option from the main menu.

3. Click the "Add Server" button.

4. Give the server a name (e.g. "The Time Vortex").

5. Enter the server IP - mc.thetimevortex.xyz - into the "Server Name" field.

6. Click "Done".

7. Select the server from the server list and click "Join Server".

Bedrock Version:

1. Launch Minecraft Bedrock Edition.

2. Click on the "Play" button from the main menu.

3. Click on the "Servers" tab.

4. Click the "Add Server" button.
5. Give the server a name (e.g. "The Time Vortex").

6. Enter the server IP - mc.thetimevortex.xyz - into the "Server Name" field.

7. Click "Done".

8. Select the server from the server list and click "Join Server".

1. Download the MC Server Connector app on your phone or tablet/ipad. Here is the Android link. Here is the IOS link.

2. Launch the app.

3. Give the server a name (e.g. "The Time Vortex").

4. Enter the server IP - mc.thetimevortex.xyz - into the "Server Address" field.
