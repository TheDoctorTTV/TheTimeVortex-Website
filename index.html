Explore, create, and survive in The Time Vortex Minecraft Server! Dive into a unique adventure with a global community on Bedrock or Java. Connect using the server IP: mc.thetimevortex.xyz [1.13 - 1.20], and unleash your Minecraft creativity and survival skills.


How to Join The Time Vortex Minecraft Server


Follow the instructions below to join The Time Vortex Minecraft server on Java and Bedrock editions.


Java Version:


  1. Launch Minecraft Java Edition.
  2. Select "Multiplayer" from the main menu.
  3. Click the "Add Server" button.
  4. Name the server as "The Time Vortex".
  5. Enter the server IP: mc.thetimevortex.xyz in the "Server Address" field.
  6. Click "Done".
  7. Select The Time Vortex from the server list and click "Join Server".

Bedrock Version:


  1. Launch Minecraft Bedrock Edition.
  2. Select "Play" from the main menu.
  3. Go to the "Servers" tab and click "Add Server".
  4. Name the server as "The Time Vortex".
  5. Enter the server IP: mc.thetimevortex.xyz and port: 8148.
  6. Click "Done".
  7. Select The Time Vortex from the server list and click "Join Server".

Bedrock Console Version:


  1. Download the MC Server Connector app on your device: Android | iOS
  2. Launch the MC Server Connector app.
  3. Name the server as "The Time Vortex".
  4. Enter the server IP: mc.thetimevortex.xyz and port: 8148.
  5. Click "Start Server". The server will appear under the Friends tab.
  6. Select The Time Vortex and click "Join".
