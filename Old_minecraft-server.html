How to Join The Time Vortex Minecraft Server

Follow the instructions below to join The Time Vortex Minecraft server on Java and Bedrock editions.

Java Version:

  1. Launch Minecraft Java Edition.
  2. Select "Multiplayer" from the main menu.
  3. Click the "Add Server" button.
  4. Name the server as "The Time Vortex".
  5. Enter the server IP: mc.thetimevortex.xyz in the "Server Address" field.
  6. Click "Done".
  7. Select The Time Vortex from the server list and click "Join Server".

Bedrock Version:

  4. Name the server as "The Time Vortex".
  5. Enter the server IP: mc.thetimevortex.xyz and port: 8148.
  6. Click "Done".
  7. Select The Time Vortex from the server list and click "Join Server".
